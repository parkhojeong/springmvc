@ResponseBody flow
- if return String,
- use HttpMessageConverter(StringHttpMessageConverter) instead viewResolver

spring mvc use HTTP message converter below cases 
- HTTP request: @RequestBody, HttpEntity(RequestEntity)
- HTTP response: @ResponseBody, HttpEntity(ResponseEntity)

HttpMessageConverter interface structure
- canRead, canWrite: check class type, media type
  - request: Content-Type header
  - response: Accept header
- read, write: convert

spring boot basic message converter
- 0 = ByteArrayHttpMessageConverter: byte[]
  - class type: byte[], media type: */*
  - request ex) @RequestBody byte[] data
  - response ex) @ResponseBody return byte[], media type: application/octet-stream
- 1 = StringHttpMessageConverter: String 
  - class type: String, media type: */*
  - request ex) @RequestBody String data
  - response ex) @ResponseBody return "ok", media type: text/plain 
- 2 = MappingJackson2HttpMessageConverter: application/json
  - class type: Object or HashMap, media type: application/json
  - request ex) @RequestBody HelloData data
  - response ex) @ResponseBody return new HelloData(), media type: application/json

HTTP request message converter
- parameter @RequestBody or HttpEntity(RequestEntity)
- MessageConverter canread()
  - support class type
    - ex) @RequestBody class type(byte[], String, HelloData)
  - support Content-Type header
    - ex) application/json, text/plain, */*
- canread() return true, read() convert request body

HTTP response message converter
- return value @ResponseBody or HttpEntity(ResponseEntity)
- MessageConverter canwrite()
  - support class type
    - ex) @ResponseBody class type(byte[], String, HelloData)
  - support Accept header(or @RequestMapping(produces))
    - ex) application/json, text/plain, */*
- canwrite() return true, write() convert response body

ex)
[StringHttpMessageConverter usage]
== StringHttpMessageConverter ex start =
content-type: application/json

@RequestMapping
void hello(@RequestBody String data) { }
== StringHttpMessageConverter ex end =
ByteArayHttpMessageConverter canread() -> String type X -> return false
[StringHttpMessageConverter] canread() -> String type O -> */* accept application/json O -> return true 

[MappingJackson2HttpMessageConverter usage]
== ex start =
accept: application/json

@RequestMapping
void hello(@RequestBody HelloData data) {}
== ex end =
ByteArayHttpMessageConverter canread() -> HelloData type X -> return false
StringHttpMessageConverter canread() -> HelloData type X -> return false
[MappingJackson2HttpMessageConverter] canread() -> HelloData type O -> application/json O -> return true
